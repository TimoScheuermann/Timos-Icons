Timo's Iconset

Free to use custom created Icons

Basic Use

Add this to your <head> and your are ready to go!

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/TimoScheuermann/Timos-Icons/docs/iconSet/style.css" />

Use them inside your HTML

<i class="ti ti-{ICONNAME}"></i> A List with all Icons can be found here Icon List

Example

<i class="ti ti-book"></i>

Result

Changing Color and Size

Its as simple as setting the fontsize and color of a text

Example

<i style="font-size: 5em; color: red" class="ti ti-like"></i>

Result

You can even add gradients

Therfore add your gradient-background to i::before

Example

i::before { background: radial-gradient( circle at 30% 107%, #fdf497 0%, #fdf497 5%, #fd5949 45%, #d6249f 60%, #285aeb 90% ); }

Result
